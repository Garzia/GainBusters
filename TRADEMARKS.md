You generally do not need explicit permission to use the GainBusters wordmark or logo in the following scenarios:

## 3. Restricted & Prohibited Uses

Without prior written authorization from Francesco Garzia, you are **strictly prohibited** from:

- **Implied Endorsement**: Using official branding elements in any way that implies official endorsement, sponsorship, or partnership with Francesco Garzia.

---

## 4. Forking Guidelines under AGPLv3

If you fork or modify the GainBusters codebase to run your own custom instance or public redistribution under the AGPLv3 license:
1. You must remove official GainBusters branding and logos from the primary user interface if significant modifications are introduced.
2. You must give your project a distinct, unique name that does not include "GainBusters".
